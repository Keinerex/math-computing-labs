## Вариант 28
**Исходные данные:** x = [0, 1, 2, 3], y = [0, 5, 6, 1]

## 1. Постановка задачи

Дана дискретная функция y(x), заданная таблицей значений (x_k, y_k), k = 0, ..., n.
Требуется:
1. Найти функцию F(x), проходящую через все заданные точки (интерполяция)
2. Найти функцию F(x), приближающую данные в смысле минимума ошибки (аппроксимация)
3. Оценить качество приближения, вычислить значение в заданной точке, экстраполировать
4. Вычислить интеграл и производную от найденной функции

## 2. Интерполяция через решение СЛАУ (алгебраическая интерполяция)

### 2.1 Постановка
Ищем алгебраический полином степени n:
```
P_n(x) = a_0 + a_1*x + a_2*x^2 + ... + a_n*x^n
```

Условие интерполяции: P_n(x_k) = y_k для всех k = 0, ..., n.

### 2.2 Система линейных уравнений
Подставляя узлы, получаем СЛАУ:
```
a_0 + a_1*x_0 + a_2*x_0^2 + ... + a_n*x_0^n = y_0
a_0 + a_1*x_1 + a_2*x_1^2 + ... + a_n*x_1^n = y_1
...
a_0 + a_1*x_n + a_2*x_n^2 + ... + a_n*x_n^n = y_n
```

В матричной форме: **V * a = y**, где V — матрица Вандермонда:
```
| 1   x_0   x_0^2  ...  x_0^n  |   | a_0 |   | y_0 |
| 1   x_1   x_1^2  ...  x_1^n  | * | a_1 | = | y_1 |
| ...                            |   | ... |   | ... |
| 1   x_n   x_n^2  ...  x_n^n  |   | a_n |   | y_n |
```

### 2.3 Теорема существования и единственности
Если все узлы x_k попарно различны, то определитель Вандермонда:
```
det(V) = prod_{0 <= i < j <= n} (x_j - x_i) != 0
```
Следовательно, СЛАУ имеет **единственное решение**.

### 2.4 Результат для варианта 28
```
P(x) = -1/3 * x^3 - x^2 + 19/3 * x = -0.3333x^3 - 1.0x^2 + 6.3333x
```
Проверка: P(0) = 0, P(1) = -1/3 - 1 + 19/3 = 5, P(2) = -8/3 - 4 + 38/3 = 6, P(3) = -9 - 9 + 19 = 1

### 2.5 Оценка погрешности
Для интерполяции погрешность в точке x:
```
R_n(x) = f^{(n+1)}(xi) / (n+1)! * prod_{k=0}^n (x - x_k)
```
где xi лежит в интервале, содержащем x и все узлы. Для таблично заданной функции производная неизвестна, поэтому точную оценку получить нельзя.

## 3. Интерполяция методом Лагранжа

### 3.1 Формула Лагранжа
Полином строится как сумма базисных полиномов:
```
L_n(x) = sum_{k=0}^n y_k * l_k(x)
```

Базисные полиномы Лагранжа:
```
l_k(x) = prod_{j=0, j!=k}^n (x - x_j) / (x_k - x_j)
```

### 3.2 Свойства базисных функций
```
l_k(x_i) = 1  если i = k
l_k(x_i) = 0  если i != k
```
То есть каждая базисная функция равна 1 в "своем" узле и 0 во всех остальных.

### 3.3 Пример для варианта 28
Для n = 3:
```
l_0(x) = (x-1)(x-2)(x-3) / ((0-1)(0-2)(0-3)) = (x-1)(x-2)(x-3) / (-6)
l_1(x) = x(x-2)(x-3) / ((1-0)(1-2)(1-3)) = x(x-2)(x-3) / 2
l_2(x) = x(x-1)(x-3) / ((2-0)(2-1)(2-3)) = x(x-1)(x-3) / (-2)
l_3(x) = x(x-1)(x-2) / ((3-0)(3-1)(3-2)) = x(x-1)(x-2) / 6
```

L(x) = 0*l_0(x) + 5*l_1(x) + 6*l_2(x) + 1*l_3(x)

Раскрывая скобки и приводя подобные, получаем тот же полином: L(x) = -1/3*x^3 - x^2 + 19/3*x

### 3.4 Достоинства и недостатки
+ Простота реализации, явная формула
+ Не нужно решать систему уравнений
- Добавление нового узла требует пересчета всех базисных функций
- При большом n возможны осцилляции (явление Рунге)
- Вычислительно неустойчив при больших n

## 4. Интерполяция методом Ньютона

### 4.1 Полином Ньютона
```
N_n(x) = f[x_0] + f[x_0,x_1]*(x-x_0) + f[x_0,x_1,x_2]*(x-x_0)(x-x_1) + ...
       + f[x_0,...,x_n]*(x-x_0)*...*(x-x_{n-1})
```

### 4.2 Разделенные разности
**Разделенная разность 1-го порядка:**
```
f[x_i, x_{i+1}] = (y_{i+1} - y_i) / (x_{i+1} - x_i)
```

**Разделенная разность k-го порядка (рекурсивно):**
```
f[x_i, ..., x_{i+k}] = (f[x_{i+1}, ..., x_{i+k}] - f[x_i, ..., x_{i+k-1}]) / (x_{i+k} - x_i)
```

### 4.3 Таблица разделенных разностей для варианта 28
```
x_i    f[x_i]    f[x_i,x_{i+1}]    f[2-го пор.]    f[3-го пор.]
----------------------------------------------------------------
0      0         5                 -2              -1/3
1      5         1                 -3
2      6         -5
3      1
```

Вычисления:
```
f[0,1] = (5-0)/(1-0) = 5
f[1,2] = (6-5)/(2-1) = 1
f[2,3] = (1-6)/(3-2) = -5

f[0,1,2] = (1-5)/(2-0) = -4/2 = -2
f[1,2,3] = (-5-1)/(3-1) = -6/2 = -3

f[0,1,2,3] = (-3-(-2))/(3-0) = -1/3
```

### 4.4 Полином
```
N(x) = 0 + 5(x-0) + (-2)(x-0)(x-1) + (-1/3)(x-0)(x-1)(x-2)
     = 5x - 2x(x-1) - 1/3*x(x-1)(x-2)
```

После раскрытия: N(x) = -1/3*x^3 - x^2 + 19/3*x

### 4.5 Достоинства и недостатки
+ Добавление нового узла: добавляется одно слагаемое, остальные не меняются
+ Удобен для экстраполяции
+ Таблица разделенных разностей дает представление о поведении функции
- Те же проблемы с устойчивостью при больших n

## 5. Интерполяция кубическими сплайнами

### 5.1 Определение сплайна
**Сплайном** на отрезке [a,b] с узлами x_0 < x_1 < ... < x_n называется функция S(x), которая:
1. На каждом [x_i, x_{i+1}] является многочленом степени 3
2. S(x), S'(x), S''(x) непрерывны на [a,b]
3. S(x_i) = y_i (условие интерполяции)

### 5.2 Представление сплайна
На каждом отрезке [x_i, x_{i+1}]:
```
S_i(x) = a_i + b_i*(x-x_i) + c_i*(x-x_i)^2 + d_i*(x-x_i)^3
```

### 5.3 Условия для определения коэффициентов
1. **Интерполяция:** S_i(x_i) = y_i, S_i(x_{i+1}) = y_{i+1}
2. **Непрерывность 1-й производной:** S_i'(x_{i+1}) = S_{i+1}'(x_{i+1})
3. **Непрерывность 2-й производной:** S_i''(x_{i+1}) = S_{i+1}''(x_{i+1})
4. **Граничные условия** (естественный сплайн): S''(x_0) = S''(x_n) = 0

### 5.4 Система для c_i
Вводя h_i = x_{i+1} - x_i, получаем трехдиагональную систему:
```
h_{i-1}*c_{i-1} + 2*(h_{i-1}+h_i)*c_i + h_i*c_{i+1} = 3*((y_{i+1}-y_i)/h_i - (y_i-y_{i-1})/h_{i-1})
```

Для равномерной сетки h_i = h:
```
c_{i-1} + 4*c_i + c_{i+1} = 3*(y_{i+1} - 2*y_i + y_{i-1})/h^2
```

### 5.5 Результат для варианта 28
```
S_0(x) на [0,1]: -0.6667x^3 + 0x^2 + 5.6667x + 0
S_1(x) на [1,2]: -0.6667(x-1)^3 - 2(x-1)^2 + 3.6667(x-1) + 5
S_2(x) на [2,3]: 1.3333(x-2)^3 - 4(x-2)^2 - 2.3333(x-2) + 6
```

### 5.6 Оценка погрешности сплайна
Для равномерной сетки:
```
|f(x) - S(x)| <= (5/384) * max|f^{(4)}(xi)| * h^4
```

### 5.7 Преимущества сплайнов
+ Отсутствие осцилляций Рунге
+ Сходимость при h -> 0
+ Гладкость (непрерывны S, S', S'')
+ Вычислительная устойчивость
+ Локальность: изменение одного узла влияет только на соседние отрезки

## 6. Метод наименьших квадратов (МНК)

### 6.1 Постановка задачи
Для табличной функции с ошибками ищем обобщенный многочлен:
```
F_m(x) = c_0*phi_0(x) + c_1*phi_1(x) + ... + c_m*phi_m(x)
```
где phi_k(x) — базисные функции (обычно степени x: 1, x, x^2, ...).

Критерий: минимизировать сумму квадратов отклонений:
```
S(c_0,...,c_m) = sum_{k=0}^n (F_m(x_k) - y_k)^2 -> min
```

### 6.2 Нормальная система уравнений
Из условия экстремума (частные производные = 0):
```
sum_{j=0}^m c_j * sum_{k=0}^n phi_j(x_k)*phi_i(x_k) = sum_{k=0}^n y_k*phi_i(x_k),   i = 0,...,m
```

Для степенного базиса phi_j(x) = x^j:
```
sum_{k=0}^n x_k^{i+j} * c_j = sum_{k=0}^n y_k * x_k^i
```

### 6.3 Частные случаи
**Прямая (m = 1):** F(x) = c_0 + c_1*x
```
c_0 * (n+1) + c_1 * sum(x_k) = sum(y_k)
c_0 * sum(x_k) + c_1 * sum(x_k^2) = sum(y_k * x_k)
```

**Парабола (m = 2):** F(x) = c_0 + c_1*x + c_2*x^2
```
c_0*(n+1) + c_1*sum(x_k) + c_2*sum(x_k^2) = sum(y_k)
c_0*sum(x_k) + c_1*sum(x_k^2) + c_2*sum(x_k^3) = sum(y_k*x_k)
c_0*sum(x_k^2) + c_1*sum(x_k^3) + c_2*sum(x_k^4) = sum(y_k*x_k^2)
```

### 6.4 Результат для варианта 28
Для m=1: F(x) = 2.4 + 0.4x, SSE = 25.2
Для m=2: F(x) = -0.1 + 7.9x - 2.5x^2, SSE = 0.2

### 6.5 Критерий выбора степени m
- Если m = n: интерполяция (SSE = 0, но переобучение)
- Если m << n: грубое приближение
- Оптимальное m определяется по SSE: m увеличивается, пока SSE не станет достаточно малым

## 7. Численное интегрирование

### 7.1 Формула Симпсона
```
integral_a^b f(x)dx = h/3 * [f(x_0) + 4*sum_{нечет} f(x_i) + 2*sum_{четет} f(x_i) + f(x_n)]
```
где h = (b-a)/n, n — четное.

**Погрешность:** R = -(b-a)*h^4/180 * f^{(4)}(xi) = O(h^4)

### 7.2 Результат для варианта 28
Для P(x) = -1/3*x^3 - x^2 + 19/3*x:
```
integral_0^3 P(x)dx = [-1/12*x^4 - 1/3*x^3 + 19/6*x^2]_0^3 = 12.75
```

## 8. Численное дифференцирование

### 8.1 Формула центральной разности
```
f'(x) = (f(x+h) - f(x-h)) / (2h) + O(h^2)
```

### 8.2 Результат для варианта 28
```
P'(x) = -x^2 - 2x + 19/3
P'(2.58) = -2.58^2 - 2*2.58 + 19/3 = -5.4831
```

## 9. Вопросы для защиты

### Почему СЛАУ, Лагранж и Ньютон дают одинаковый результат?
По теореме единственности алгебраической интерполяции: для n+1 различного узла существует единственный полином степени <= n. Все три метода находят этот единственный полином.

### В чем разница между интерполяцией и аппроксимацией?
- **Интерполяция:** полином проходит точно через все узлы (SSE = 0)
- **Аппроксимация (МНК):** полином минимизирует общую ошибку, но не обязан проходить через узлы. Применяется, когда данные содержат ошибки.

### Что такое разделенные разности и что они аппроксимируют?
Разделенная разность k-го порядка аппроксимирует k-ю производную функции с точностью до числового множителя k!

### Почему сплайны лучше высокостепенных полиномов?
Высокостепенные полиномы страдают от осцилляций Рунге. Сплайны — кусочные функции низкой степени, поэтому не дают больших осцилляций и обеспечивают сходимость.

### Какая погрешность у формулы Симпсона?
O(h^4) — очень высокая точность. Для полинома 3-й степени дает точный результат.

- P(x) = -1/3*x^3 - x^2 + 19/3*x
- P(2.58) = 3.959
- integral_0^3 P(x)dx = 12.75
- P'(2.58) = -5.483
