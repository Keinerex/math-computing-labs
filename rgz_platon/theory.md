## 1. Постановка задачи

Дана дискретная функция y(x) в узлах x_k = [0, 1, 2, 3] со значениями y_k = [0, 4, 5, 2].
Требуется:
1. Найти интерполяционный полином, проходящий через все узлы
2. Найти аппроксимирующую функцию методом наименьших квадратов
3. Оценить качество приближения, найти значение в точке x = 2.58
4. Выполнить экстраполяцию на x = 4 и x = 5
5. Вычислить интеграл на [0; 3] и производную в точке x = 2.58

## 2. Интерполяция через решение СЛАУ

### 2.1 Алгебраическая интерполяция
Ищем полином P(x) = a_0 + a_1*x + a_2*x^2 + a_3*x^3 степени n = 3.

Условие интерполяции: P(x_k) = y_k для всех узлов.

### 2.2 Матрица Вандермонда
```
| 1   x_0   x_0^2   x_0^3 |   | a_0 |   | y_0 |
| 1   x_1   x_1^2   x_1^3 | * | a_1 | = | y_1 |
| 1   x_2   x_2^2   x_2^3 |   | a_2 |   | y_2 |
| 1   x_3   x_3^2   x_3^3 |   | a_3 |   | y_3 |
```

Для варианта 11:
```
| 1   0   0   0 |   | a_0 |   | 0 |
| 1   1   1   1 | * | a_1 | = | 4 |
| 1   2   4   8 |   | a_2 |   | 5 |
| 1   3   9  27 |   | a_3 |   | 2 |
```

### 2.3 Решение системы
Из первого уравнения: a_0 = 0.

Подставляем a_0 = 0 в остальные:
```
a_1 + a_2 + a_3 = 4          (1)
2*a_1 + 4*a_2 + 8*a_3 = 5   (2)
3*a_1 + 9*a_2 + 27*a_3 = 2  (3)
```

Из (1): a_1 = 4 - a_2 - a_3

Подставляем в (2):
```
2*(4 - a_2 - a_3) + 4*a_2 + 8*a_3 = 5
8 - 2*a_2 - 2*a_3 + 4*a_2 + 8*a_3 = 5
8 + 2*a_2 + 6*a_3 = 5
2*a_2 + 6*a_3 = -3              (4)
```

Подставляем a_1 в (3):
```
3*(4 - a_2 - a_3) + 9*a_2 + 27*a_3 = 2
12 - 3*a_2 - 3*a_3 + 9*a_2 + 27*a_3 = 2
12 + 6*a_2 + 24*a_3 = 2
6*a_2 + 24*a_3 = -10
3*a_2 + 12*a_3 = -5             (5)
```

Из (4): a_2 = (-3 - 6*a_3)/2

Подставляем в (5):
```
3*(-3 - 6*a_3)/2 + 12*a_3 = -5
(-9 - 18*a_3)/2 + 12*a_3 = -5
-4.5 - 9*a_3 + 12*a_3 = -5
3*a_3 = -0.5
a_3 = -1/6
```

Тогда:
```
a_2 = (-3 - 6*(-1/6))/2 = (-3 + 1)/2 = -1
a_1 = 4 - (-1) - (-1/6) = 4 + 1 + 1/6 = 31/6 = 5.1667
```

**Ответ: P(x) = -1/6*x^3 - x^2 + 31/6*x**

### 2.4 Проверка
```
P(0) = 0                                    = 0   ✓
P(1) = -1/6 - 1 + 31/6 = 30/6 - 1 = 4      = 4   ✓
P(2) = -8/6 - 4 + 62/6 = 54/6 - 4 = 5      = 5   ✓
P(3) = -27/6 - 9 + 93/6 = 66/6 - 9 = 2     = 2   ✓
```

## 3. Интерполяция методом Лагранжа

### 3.1 Базисные полиномы
```
l_0(x) = (x-1)(x-2)(x-3) / ((0-1)(0-2)(0-3)) = (x-1)(x-2)(x-3) / (-6)
l_1(x) = x(x-2)(x-3) / ((1-0)(1-2)(1-3)) = x(x-2)(x-3) / 2
l_2(x) = x(x-1)(x-3) / ((2-0)(2-1)(2-3)) = x(x-1)(x-3) / (-2)
l_3(x) = x(x-1)(x-2) / ((3-0)(3-1)(3-2)) = x(x-1)(x-2) / 6
```

### 3.2 Интерполяционный полином
```
L(x) = 0*l_0(x) + 4*l_1(x) + 5*l_2(x) + 2*l_3(x)
     = 2*x(x-2)(x-3) - 2.5*x(x-1)(x-3) + (1/3)*x(x-1)(x-2)
```

После раскрытия скобок и приведения подобных членов:
**L(x) = -1/6*x^3 - x^2 + 31/6*x** — тот же результат, что и СЛАУ.

### 3.3 Проверка свойства базисных функций
```
l_0(0) = 1, l_0(1) = l_0(2) = l_0(3) = 0
l_1(1) = 1, l_1(0) = l_1(2) = l_1(3) = 0
l_2(2) = 1, l_2(0) = l_2(1) = l_2(3) = 0
l_3(3) = 1, l_3(0) = l_3(1) = l_3(2) = 0
```

## 4. Интерполяция методом Ньютона

### 4.1 Таблица разделенных разностей

**Разделенные разности 1-го порядка:**
```
f[0,1] = (y_1 - y_0)/(x_1 - x_0) = (4 - 0)/(1 - 0) = 4
f[1,2] = (y_2 - y_1)/(x_2 - x_1) = (5 - 4)/(2 - 1) = 1
f[2,3] = (y_3 - y_2)/(x_3 - x_2) = (2 - 5)/(3 - 2) = -3
```

**Разделенные разности 2-го порядка:**
```
f[0,1,2] = (f[1,2] - f[0,1])/(x_2 - x_0) = (1 - 4)/(2 - 0) = -3/2 = -1.5
f[1,2,3] = (f[2,3] - f[1,2])/(x_3 - x_1) = (-3 - 1)/(3 - 1) = -4/2 = -2
```

**Разделенная разность 3-го порядка:**
```
f[0,1,2,3] = (f[1,2,3] - f[0,1,2])/(x_3 - x_0) = (-2 - (-1.5))/(3 - 0) = -0.5/3 = -1/6
```

### 4.2 Полная таблица
```
xi   f[xi]   f[xi,xi+1]   f[2-го пор]   f[3-го пор]
---------------------------------------------------
0    0       4            -1.5          -1/6
1    4       1            -2            —
2    5       -3           —             —
3    2       —            —             —
```

### 4.3 Полином Ньютона
```
N(x) = f[0] + f[0,1]*(x-0) + f[0,1,2]*(x-0)(x-1) + f[0,1,2,3]*(x-0)(x-1)(x-2)
     = 0 + 4*x + (-1.5)*x*(x-1) + (-1/6)*x*(x-1)*(x-2)
     = 4x - 1.5x^2 + 1.5x - (1/6)*x*(x^2 - 3x + 2)
     = 5.5x - 1.5x^2 - 1/6*x^3 + 0.5x^2 - 1/3*x
     = -1/6*x^3 - x^2 + 31/6*x
```

## 5. Интерполяция кубическими сплайнами

### 5.1 Построение естественного кубического сплайна

На каждом отрезке [x_i, x_{i+1}] сплайн имеет вид:
```
S_i(x) = a_i + b_i*(x-x_i) + c_i*(x-x_i)^2 + d_i*(x-x_i)^3
```

**Шаг 1: Значения a_i = y_i**
```
a_0 = 0, a_1 = 4, a_2 = 5
```

**Шаг 2: Система для c_i**
Для естественного сплайна (c_0 = c_3 = 0) и равномерной сетки h = 1:
```
4*c_1 + c_2 = 3*(a_2 - 2*a_1 + a_0) = 3*(5 - 8 + 0) = -9
c_1 + 4*c_2 = 3*(a_3 - 2*a_2 + a_1) = 3*(2 - 10 + 4) = -12
```

Решение: c_1 = -1.6, c_2 = -2.6

**Шаг 3: Вычисление b_i и d_i**
```
b_i = (a_{i+1} - a_i)/h - h*(2*c_i + c_{i+1})/3
d_i = (c_{i+1} - c_i)/(3*h)
```

b_0 = 4 - (2*0 + (-1.6))/3 = 4 + 1.6/3 = 4.5333
d_0 = (-1.6 - 0)/3 = -0.5333

b_1 = 1 - (2*(-1.6) + (-2.6))/3 = 1 + 5.8/3 = 2.9333
d_1 = (-2.6 - (-1.6))/3 = -1/3 = -0.3333

b_2 = -3 - (2*(-2.6) + 0)/3 = -3 + 5.2/3 = -1.2667
d_2 = (0 - (-2.6))/3 = 0.8667

**Результат:**
```
S_0(x) на [0,1]: -0.5333x^3 + 0x^2 + 4.5333x + 0
S_1(x) на [1,2]: -0.3333(x-1)^3 - 1.6(x-1)^2 + 2.9333(x-1) + 4
S_2(x) на [2,3]: 0.8667(x-2)^3 - 2.6(x-2)^2 - 1.2667(x-2) + 5
```

## 6. Метод наименьших квадратов

### 6.1 МНК с m = 2 (парабола)
Ищем F(x) = c_0 + c_1*x + c_2*x^2.

**Вычисляем суммы:**
```
n = 4
sum(x) = 0+1+2+3 = 6
sum(x^2) = 0+1+4+9 = 14
sum(x^3) = 0+1+8+27 = 36
sum(x^4) = 0+1+16+81 = 96
sum(y) = 0+4+5+2 = 11
sum(y*x) = 0+4+10+6 = 20
sum(y*x^2) = 0+4+20+18 = 42
```

**Нормальная система:**
```
4*c_0 + 6*c_1 + 14*c_2 = 11
6*c_0 + 14*c_1 + 36*c_2 = 20
14*c_0 + 36*c_1 + 96*c_2 = 42
```

Решение: c_0 = -0.05, c_1 = 5.95, c_2 = -1.75

**F(x) = -1.75x^2 + 5.95x - 0.05**

### 6.2 Сумма квадратов отклонений
```
SSE = sum(F(x_k) - y_k)^2
F(0) = -0.05,  ошибка = 0.0025
```

## 8. Вопросы для защиты

**1. Почему СЛАУ, Лагранж и Ньютон дают одинаковый результат?**
Теорема единственности: для n+1 различного узла существует единственный полином степени <=n, проходящий через все узлы.

**2. В чем разница между интерполяцией и аппроксимацией?**
Интерполяция: точное прохождение через узлы. Аппроксимация (МНК): минимизация общей ошибки, полином не обязан проходить через узлы.

**3. Что такое разделенные разности?**
Обобщение производных. f[x_i,...,x_{i+k}] аппроксимирует f^{(k)}(xi)/k!

**4. Почему сплайны лучше высокостепенных полиномов?**
Нет осцилляций Рунге, локальность, сходимость.

**5. Интерполяционный полином варианта 11:**

**6. Интеграл и производная:**
