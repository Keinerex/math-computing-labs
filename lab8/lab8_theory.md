## 1. Основные понятия

### Задача оптимизации
Найти экстремум (минимум или максимум) функции f(x) на заданном множестве.

### Типы задач
- **Безусловная оптимизация**: минимизировать f(x) без ограничений
- **Условная оптимизация**: минимизировать f(x) при ограничениях g_i(x) <= 0, h_j(x) = 0

### Унимодальная функция
Функция унимодальна на [a,b], если имеет единственный экстремум и монотонна по обе стороны от него.

## 2. Методы одномерной минимизации

### 2.1 Метод дихотомии
Делим отрезок пополам, сравниваем значения функции в двух точках около середины, отбрасываем половину отрезка.

- За одну итерацию интервал уменьшается в ~2 раза
- На каждой итерации 2 вычисления функции
- Число итераций: n >= ln((b-a)/epsilon) / ln(2)

### 2.2 Метод золотого сечения
Пробные точки делят отрезок в отношении золотого сечения tau = (sqrt(5)-1)/2 = 0.618.

```
x1 = a + (1-tau)*(b-a)
x2 = a + tau*(b-a)
```

- За одну итерацию интервал уменьшается в tau ~ 1.618 раз
- Одна точка переиспользуется (1 вычисление функции на шаг, кроме первого)
- Число итераций: n >= ln((b-a)/epsilon) / ln(1/tau)

### 2.3 Метод Фибоначчи
Использует числа Фибоначчи для выбора пробных точек.

```
F(0)=1, F(1)=1, F(n)=F(n-1)+F(n-2)
x1 = a + F(n-k-1)/F(n-k+1) * (b-a)
```

- Теоретически оптимален по числу вычислений функции
- Требует предварительного определения числа шагов
- Похож на золотое сечение, но с переменным отношением

### Сравнение методов одномерной оптимизации

| Метод | Редукция интервала | Вычислений f/итер | Преимущества |
|-------|-------------------|-------------------|--------------|
| Дихотомия | ~2 раза | 2 | Простота |
| Золотое сечение | ~1.618 раз | 1 (после 1-й) | Не нужно знать n заранее |
| Фибоначчи | ~1.618 раз | 1 (после 1-й) | Оптимален по числу вычислений |

## 3. Поиск глобального экстремума

Для многоэкстремальных функций локальные методы находят только один экстремум.

**Метод сканирования**: разбиваем [a,b] на N подынтервалов, на каждом ищем локальный экстремум, выбираем лучший.

## 4. Многомерная оптимизация

### Градиентный метод (метод наискорейшего спуска)
```
x^{k+1} = x^k - alpha * grad(f(x^k))
```
Для максимизации: идем в направлении градиента (а не антиградиента).

### Условная оптимизация

#### Метод штрафов
Преобразуем задачу условной оптимизации в последовательность задач безусловной:
```
F(x, r) = f(x) + r * P(x)
P(x) = sum(max(0, g_i(x))^2) + sum(h_j(x)^2)
```
r увеличивается на каждой итерации (r_{k+1} = C * r_k, C > 1).

#### Метод барьеров
Используется для ограничений-неравенств. Барьерная функция:
```
F(x, r) = f(x) - r * sum(1/g_i(x))   (обратный барьер)
```
Точки остаются внутри допустимой области. r уменьшается.

## 5. Вопросы для защиты

### Для каких функций эффективны методы дихотомии, золотого сечения, Фибоначчи?
Для унимодальных функций на отрезке [a,b].

### В чем отличие метода штрафов от метода барьеров?
- Штрафы: начальная точка ВНЕ допустимой области, штраф за нарушение, r увеличивается
- Барьеры: начальная точка ВНУТРИ области, барьер на границе, r уменьшается

### Что такое унимодальная функция?
Функция с единственным экстремумом на отрезке, монотонная по обе стороны.

### Алгоритм градиентного спуска?
x^{k+1} = x^k - alpha * grad(f(x^k)). Шаг alpha выбирается из условия убывания функции.

### В чем идея метода штрафных функций?
Преобразование задачи условной оптимизации в последовательность задач безусловной путем добавления штрафа за нарушение ограничений.

### Зачем увеличивается коэффициент штрафа?
Чтобы на последующих итерациях решение все ближе приближалось к допустимой области.

### Чем отличаются методы Фибоначчи и золотого сечения?
Фибоначчи использует переменное отношение (числа Фибоначчи), золотое сечение - постоянное (0.618).

### Как найти глобальный экстремум?
Разбиение области на подынтервалы + поиск локальных экстремумов на каждом + выбор лучшего.

### Для варианта 20: какое точное решение?
- Одномерная: x_min = pi, f_min = -1
- Многомерная: x1 = -9/8 = -1.125, x2 = -7/8 = -0.875, f_max = 6.0625

### Какая точность достигнута методом Фибоначчи?
|x_min - pi| = 6.78e-05 (при epsilon = 1e-4)
