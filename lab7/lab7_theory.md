# ЛР7: Численное решение ОДУ - Теория для защиты

## Вариант 20
y' = 21y - 8y^3 - x^2 * y,  y(0) = 1,  x in [0, 1]

---

## 1. Основные понятия

### Задача Коши для ОДУ 1-го порядка
Найти функцию y(x), удовлетворяющую:
- Дифференциальному уравнению: y' = f(x, y)
- Начальному условию: y(x0) = y0

### Численное решение
Нахождение приближенных значений y(x) в дискретных точках (сетке) x0, x1, ..., xn.

---

## 2. Метод Эйлера (явный)

### Формула
y_{k+1} = y_k + h * f(x_k, y_k)

где h - шаг интегрирования, h = (b-a)/N

### Геометрическая интерпретация
На каждом шаге продолжаем решение по касательной в точке (x_k, y_k). Наклон касательной = f(x_k, y_k).

### Погрешность
- Локальная (на шаге): O(h^2)
- Глобальная (на интервале): O(h)

### Устойчивость
Коэффициент усиления погрешности:
```
g = 1 + h * (df/dy)
```
Условие устойчивости: |g| <= 1

Для нашего уравнения:
```
df/dy = 21 - 24y^2 - x^2
g = 1 + h * (21 - 24y^2 - x^2)
```

При y(0)=1, x=0: df/dy = 21 - 24 = -3, g = 1 - 3h
|g| <= 1 => 0 <= 3h <= 2 => h <= 2/3

### Достоинства и недостатки
+ Простота реализации
+ Малые вычислительные затраты
- Низкая точность (1-й порядок)
- Условная устойчивость

---

## 3. Неявный метод Эйлера

### Формула
y_{k+1} = y_k + h * f(x_{k+1}, y_{k+1})

Правая часть зависит от y_{k+1} - неявное уравнение!

### Решение
Для нашего уравнения:
```
y_{k+1} = y_k + h * (21*y_{k+1} - 8*y_{k+1}^3 - x_{k+1}^2 * y_{k+1})
```
Это нелинейное уравнение. Решаем методом Ньютона:
```
F(y) = y - y_k - h*f(x_{k+1}, y) = 0
F'(y) = 1 - h*(df/dy)
y^{new} = y - F(y)/F'(y)
```

### Погрешность
- Локальная: O(h^2)
- Глобальная: O(h)

### Устойчивость
Коэффициент усиления:
```
g = 1 / (1 - h * df/dy)
```

Для df/dy < 0 (убывающие решения): |g| < 1 при ЛЮБОМ h
=> Безусловная устойчивость!

### Достоинства и недостатки
+ Безусловная устойчивость
- На каждом шаге нужно решать нелинейное уравнение
- Низкая точность (1-й порядок)

---

## 4. Метод Рунге-Кутта 4-го порядка (RK4)

### Формулы
```
k1 = h * f(x_k, y_k)
k2 = h * f(x_k + h/2, y_k + k1/2)
k3 = h * f(x_k + h/2, y_k + k2/2)
k4 = h * f(x_k + h, y_k + k3)

y_{k+1} = y_k + (k1 + 2*k2 + 2*k3 + k4) / 6
```

### Идея
Вычисляем производную в 4 точках шага и берем взвешенное среднее.
Эквивалентно интерполяции полиномом 4-й степени.

### Погрешность
- Локальная: O(h^5)
- Глобальная: O(h^4)

### Устойчивость
Условная устойчивость, но устойчив при значительно больших шагах, чем метод Эйлера.

### Достоинства и недостатки
+ Высокая точность (4-й порядок)
+ Можно использовать большие шаги
+ Прост в реализации (явный метод)
- Больше вычислений на шаг (4 вычисления f)
- Условная устойчивость

---

## 5. Сравнение методов

| Метод | Порядок | Погрешность | Устойчивость | Вычислений f на шаг |
|-------|---------|-------------|--------------|---------------------|
| Эйлер явный | 1 | O(h) | Условная | 1 |
| Эйлер неявный | 1 | O(h) | Безусловная | Ньютон |
| RK4 | 4 | O(h^4) | Условная | 4 |

## 6. Вопросы для защиты

### Что такое задача Коши?
Задача нахождения решения ОДУ y' = f(x,y) с начальным условием y(x0) = y0.

### В чем суть метода Эйлера?
Линейная экстраполяция по касательной: y_{k+1} = y_k + h*f(x_k, y_k).

### В чем отличие явного и неявного методов Эйлера?
В явном методе y_{k+1} вычисляется напрямую по формуле.
В неявном - y_{k+1} входит в обе части уравнения, требуется решать нелинейное уравнение.

### Почему неявный метод безусловно устойчив?
Коэффициент g = 1/(1 - h*df/dy). При df/dy < 0: |g| < 1 при любом h > 0.

### В чем суть метода Рунге-Кутта?
Многоэтапный метод: вычисляем производные в нескольких точках шага и усредняем с весами.

### Как повысить точность численного решения ОДУ?
1. Уменьшить шаг h
2. Использовать метод более высокого порядка (RK4)
3. Использовать методы с автоматическим выбором шага

### Какое уравнение дано в варианте 20?
y' = 21y - 8y^3 - x^2*y, y(0) = 1

### Как найти df/dy для варианта 20?
df/dy = 21 - 24y^2 - x^2

### Как исследовать устойчивость?
Вычисляем g = 1 + h*df/dy (явный) или g = 1/(1-h*df/dy) (неявный).
Если |g| <= 1 - метод устойчив.
